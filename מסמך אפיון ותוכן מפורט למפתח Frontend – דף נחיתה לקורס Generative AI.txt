# מסמך אפיון ותוכן מפורט למפתח Frontend – דף נחיתה לקורס Generative AI

## 1. הגדרות תשתית ועיצוב (Global Setup)

* **טכנולוגיה מומלצת:** React / Next.js בשילוב Tailwind CSS.
* **ניהול מצב (Theme):** חובה להטמיע תמיכה מלאה במצב כהה (Dark Mode) ומצב בהיר (Light Mode). **מצב ברירת המחדל יהיה Dark Mode**, כדי להעצים את הניגודיות ולהבליט את התוצרים הוויזואליים של ה-AI.
* **ריבוי שפות (i18n):** יש להגדיר תשתית לשפות (עברית/אנגלית). האתר יכלול כפתור Toggle המחליף בין השפות ומשנה אוטומטית את כיווניות האתר (RTL לעברית, LTR לאנגלית). **כל הטקסטים במסמך זה מובאים בעברית ויש להזין אותם לקובץ תרגום (למשל `he.json`).**
* **אנימציות:** יש להשתמש ב-Fade-ins עדינים בחשיפת אלמנטים בגלילה (Scroll Reveal) כדי לשדר תחכום.

## 2. מבנה הדף והתוכן הטקסטואלי המלא (Component by Component)

### א. Header (סרגל ניווט עליון - Sticky)

* **לוגו:** מיקום לוגו הקורס (מתוך תיקיית `logo/` במאגר).
* **כפתורי שליטה (Toggles):** * כפתור שמש/ירח (Theme).
* כפתור EN/HE (Language).


* **ניווט מהיר פנימי (Anchor Links):**
* התפיסה שלנו
* מה תלמדו
* דרישות קדם
* צור קשר (כפתור מודגש בפינה)



### ב. Hero Section (המסך הראשון)

* **קונספט עיצובי:** מסך חצוי (Split Screen) או שילוב אנימטיבי. צד אחד מציג שורות קוד נקיות ומינימליסטיות בפייתון (בסגנון עורך קוד כהה), ש"זורמות" או הופכות בצד השני ליצירת אמנות דינמית ומרהיבה שנוצרה ב-AI.
* **תוכן טקסטואלי:**
* **כותרת ראשית (H1):** אל תסתפקו בכלים של אחרים. תלמדו לקודד את החזון שלכם עם Generative AI.
* **תת-כותרת (H2):** קורס מעשי של 12 שבועות, בקבוצות בוטיק של עד 5 משתתפים. מהבנת קוד בסיסי בפייתון ועד ליצירת יישומי AI מורכבים, ממשקים עצמאיים, ויצירת טקסט, אודיו, וידאו ותמונות מרהיבות.
* **כפתור קריאה לפעולה ראשי (CTA):** `לפרטים ובדיקת התאמה – דברו איתי בוואטסאפ` (לחיצה תוביל לסקשן יצירת הקשר בתחתית, או ישירות ללינק וואטסאפ).



### ג. סקשן "המעבר ממשתמש ליוצר" (למה דווקא הקורס הזה?)

* **קונספט עיצובי:** 3 כרטיסיות (Cards) נקיות עם אייקונים מינימליסטיים.
* **כותרת הסקשן (H2):** למה ללמוד לבנות לבד?
* **תוכן הכרטיסיות:**
1. **כותרת:** שחרור מתלות בכלים קיימים.
**טקסט:** משתמשי קצה מוגבלים לממשקים שחברות אחרות בנו (Wrappers). כאן תלמדו את הארכיטקטורה שמאחורי הקלעים ותבנו את כלי ה-AI הייעודיים שלכם.
2. **כותרת:** קוד ככלי יצירתי.
**טקסט:** פייתון היא לא רק שפת תכנות, היא המכחול החדש שלכם. תלמדו קוד ברמה בסיסית ופרקטית כדי לחבר ממשקי API ולהפיח חיים ברעיונות מורכבים.
3. **כותרת:** שליטה מוחלטת בתוצר.
**טקסט:** מיצירת תמונות ואודיו מציאותיים ועד מודלים מותאמים אישית (LoRA) ווידאו. קבלו את הבסיס הארכיטקטוני ליצירת פרוייקטים חדשניים משלכם.



### ד. סקשן הסילבוס (מה תלמדו)

* **קונספט עיצובי:** רכיב "אקורדיון" (Accordion) נפתח. כל שורה מציגה את שם הפרק ותגית קטנה (Badge) המדגישה את *התוצר* או *היכולת* שנרכשת באותו פרק.
* **כותרת הסקשן (H2):** תכנית הלימודים - מהמסד ועד לקנבס
* **תוכן האקורדיון:**
* **חלק 1: מבוא לפייתון** * *תגית:* `הבסיס לבניית כלים עצמאיים`
* *טקסט פנימי:* משתנים, סוגי נתונים, לולאות, פונקציות, ועבודה עם קבצים וממשקי API (Requests).


* **חלק 2: עבודה עם OpenAI API**
* *תגית:* `שליטה מלאה במנוע`
* *טקסט פנימי:* יצירת טקסט, יצירת תמונות, המרת דיבור לטקסט וטקסט לדיבור (TTS/STT), יכולות ראייה ממוחשבת, שרשור בקשות וקבלת פלטים מובנים.


* **חלק 3+4: הנדסת פקודות (Claude & Perplexity)**
* *תגית:* `אמנות התקשורת עם מכונות`
* *טקסט פנימי:* עקרונות הנדסת פקודות מתקדמת, ניהול תפקידים, עבודה עם מסמכים מורכבים (PDF), תמלול, והזרמת נתונים בזמן אמת.


* **חלק 5: יצירה יצירתית עם Flux, LoRA ווידאו**
* *תגית:* `הלב הפועם של האמנות`
* *טקסט פנימי:* עבודה עם מודל Flux ליצירת תמונות באיכות קצה, כוונון עדין (Fine-Tuning) באמצעות מודלי LoRA, וטכניקות מתקדמות ליצירת וידאו.


* **חלק 6+7: כלי תכנות מבוססי AI ובניית ממשקים**
* *תגית:* `הפרויקט קורם עור וגידים`
* *טקסט פנימי:* שימוש ב-GitHub Copilot ו-Cline לכתיבת קוד מהירה. בניית ממשקי משתמש אינטראקטיביים וצ'אטבוטים בקלות באמצעות סביבת Gradio.


* **חלק 8: פרויקט גמר**
* *תגית:* `תיק העבודות שלכם`
* *טקסט פנימי:* בניית פתרון Generative AI מלא, מאפס ועד ממשק עובד, המשלב את כל הכלים שנלמדו בקורס.
